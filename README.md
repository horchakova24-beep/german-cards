# 🇩🇪 Мой Немецкий Блокнот / Mein Deutsches Notizbuch

Интерактивное веб-приложение для изучения живого, современного разговорного немецкого языка (**Alltagssprache**). 

Проект разработан с заботой о тех, кто хочет учить реальный немецкий язык, на котором говорят на улицах Германии прямо сейчас, избегая устаревших книжных штампов.

---

## ✨ Основные фичи проекта

*   **Интеграция с ИИ (Google Gemini):** Все разговорные фразы генерируются моделью `gemini-flash-latest` в реальном времени под каждую жизненную ситуацию.
*   **Автоматизация через GitHub Actions:** Обновление базы фраз происходит одной кнопкой на GitHub — робот сам делает запрос к ИИ, чистит данные и встраивает их в сайт.
*   **Современный сленг:** ИИ настроен генерировать актуальную лексику, отсекая сухой канцелярит.
*   **Визуальный прогресс:** Интерактивные карточки с цветовой кодировкой родов существительных (мужской, женский, средний) и полосой прогресса изучения.

---

## 🛠️ Технологический стек

*   **Frontend:** Чистый HTML5, CSS3 (flexbox/grid), JavaScript (ES6+).
*   **Backend / Automation:** Python 3 (для работы с API Gemini и парсинга данных), GitHub Actions.
*   **ИИ:** Google Gemini API.

## 🧭 Список разговорных тем

Проект включает в себя 10 ключевых жизненных сценариев:
1.  **Погода и планы** (`weather`) ☀️
2.  **Кафе и ресторан** (`restaurant`) ☕
3.  **Ориентация в городе** (`city`) 🗺️
4.  **Покупки и супермаркет** (`shopping`) 🛒
5.  **Самочувствие и врач** (`health`) 🩺
6.  **Поиск жилья / Отель** (`hotel`) 🔑
7.  **Работа и резюме** (`job`) 💼
8.  **Хобби и свободное время** (`hobby`) ⚽
9.  **В гостях / Праздники** (`guests`) 🎉
10. **Проблемы и ЧС** (`emergency`) 🚨

## 🚀 Как запустить генерацию новых фраз

Любой желающий (у кого есть доступ к репозиторию и настроен API-ключ) может обновить базу фраз в один клик:

1. Перейдите во вкладку **Actions** в верхнем меню репозитория.
2. В левой колонке выберите воркфлоу **Generate German Cards**.
3. Справа нажмите на серую кнопку **Run workflow** ⚙️.
4. В появившемся поле введите (или выберите) нужную тему на английском (например, `weather`, `city`, `restaurant` и т.д.).
5. Нажмите зелёную кнопку **Run workflow** для запуска робота.

Через 1-2 минуты робот закончит работу, и ваш сайт обновится автоматически!
